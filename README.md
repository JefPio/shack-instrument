# Shack Instrument (ESP32)

Modular open-source instrument for radio amateur shacks.

## Features
- ESP32 based
- GPS (NEO-6M)
- BME280 (temperature, humidity, pressure)
- Magnetic compass (HMC5883L)
- OLED 0.96" display
- Dual-button instrument-style UI
- Hierarchical information display
- Environmental and safety alerts

## Hardware
- ESP32
- OLED I2C 128x64
- GPS NEO-6M
- BME280
- HMC5883L

## Philosophy
This project is designed to work out-of-the-box using predefined screens,
while still allowing advanced users to fully customize information layout,
priority and alerts.
